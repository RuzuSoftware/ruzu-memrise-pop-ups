Ruzu Memrise pop-ups

Getting Started

  1. Ensure that you are logged in to memrise.com
  2. Click the icon to open the options (Options can also be accessed from chrome://extensions/)
  3. Select a course from the dropdown list
  4. Enable test notifications by ticking the box that looks like this:
  5. Click 'Save' to save your changes

Other Features

  1. You can change the test frequency using the 'Test frequency' dropdown
  2. Press the 'Show next question' button to force the next question pop-up to be displayed
  3. Only due questions will be reviewed by default, untick the 'Due questions only' tick box in order to review all questions regardless if they are due or not.
  4. Use 'Review type' dropdown to select one of the review types
    • All - Review all questions.
    • Difficult - Review questions marked as difficult (even without a premium memrise account).
    • Starred - Review starred questions (premium memrise account required in order to star questions).
  5. You can combine due questions with review type (eg all, due only, difficult and due, all difficult questions etc)
  6. Click the question text to go back a step if you initially click the wrong button by accident
  7. Keyboard shortcuts
    • You can enable / disable pop-ups using the keyboard shortcut (Default: Ctrl+Shift+1)
    • You can show the next question early using the keyboard shortcut (Default: Ctrl+Shift+2)
    • Visit chrome://extensions/ and click 'Keyboard shortcuts' (at the bottom of the page) to configure shortcuts
