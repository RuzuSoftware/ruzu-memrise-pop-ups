Ruzu Memrise pop-ups

Getting Started

  1. Ensure that you are logged in to memrise.com
  2. Click the icon to open the options (Options can also be accessed from chrome://extensions/)
  3. Select a course from the dropdown list
  4. Enable test notifications by ticking the box that looks like this:
  5. Click 'Save' to save your changes

Other Features

  1. Tick 'Submit answers to memrise' if you want to sync your progress to you profile on memrise.com
  2. You can also change the test frequency using the 'Test frequency' dropdown
  3. Press the 'Show next question' button to force the next question pop-up to be displayed
  4. You can enable / disable pop-ups using the keyboard shortcut (Default: Ctrl+Shift+1)
  5. You can show the next question early using the keyboard shortcut (Default: Ctrl+Shift+2)
  6. Visit chrome://extensions/ and click 'Keyboard shortcuts' (at the bottom of the page) to configure shortcuts
  7. Click the question text to go back a step if you initially click the wrong button by accident
